⚠️ ALERTE SÉCURITÉ

Tu viens de cliquer sur un lien dangereux

Cette page t’explique pourquoi ce type de lien est dangereux et comment éviter de te faire arnaquer sur Internet.

❗ Pourquoi ce lien peut être dangereux ?

Un lien peut être utilisé pour :

- voler tes comptes (YouTube, Discord, Roblox…)
- te faire entrer ton mot de passe sur un faux site
- installer un virus ou extension malveillante
- copier un site officiel pour te tromper
🧠 Comment les arnaques fonctionnent ?

Les attaques jouent sur :

- la curiosité (“cadeaux gratuits”, “robux”, “hack”)
- la peur (“ton compte va être supprimé”)
- la confiance (faux amis / faux youtubeurs)

👉 Le but est de te faire cliquer sans réfléchir.
🛡️ Comment te protéger ?

✔ Ne clique jamais sur un lien inconnu
✔ Vérifie toujours l’adresse du site (URL exacte)
✔ Ne tape jamais ton mot de passe via un lien reçu
✔ Si doute → quitte immédiatement la page
✔ Utilise un outil de vérification

⚠️ Alerte de sécurité

Tu viens de cliquer sur un lien dangereux

Ce type de page est une simulation de prévention pour t’expliquer comment les attaques sur internet fonctionnent et comment éviter de te faire piéger.

❗ Pourquoi ce lien peut être dangereux ?

Un lien dangereux peut sembler normal, mais il peut être utilisé pour :

- voler tes identifiants (YouTube, Discord, Roblox…)
- te faire entrer ton mot de passe sur un faux site
- installer un virus ou extension malveillante
- rediriger vers une copie d’un site officiel

🧠 Comment les arnaques fonctionnent

Les attaques utilisent souvent :

- la curiosité (“free robux”, “cadeau”, “hack”…)
- la peur (“ton compte va être supprimé”)
- la confiance (faux youtubeurs ou amis)

👉 Le but est simple : te faire cliquer sans réfléchir.

📌 Étapes pour te protéger

  1. Ne clique jamais sur un lien reçu par message si tu n’es pas sûr
  2. Vérifie toujours l’adresse du site (URL exacte)
  3. Ne tape jamais ton mot de passe via un lien reçu
  4. Si doute → quitte la page immédiatement
  5. Utilise un outil de vérification de liens
